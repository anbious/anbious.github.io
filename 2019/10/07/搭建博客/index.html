搭建博客

发表于 2019-10-07 分类于 杂项

本文记录了利用Github pages + hexo + nexT建站的经过，偶有踩坑。。且没有解决。。以后还是要再多试试。不过从0到1建站还是很有成就感的，记录一哈~

第一步：本地前提准备

  1. 首先推荐官方文档，懒人可参考一下步骤

  2. 下载nodejs，Windows用户官网下载，LTS版本，然后安装，默认会配置路径，别忘了重启一下终端，再不行就重启电脑=。=

  3. 选择一个自己心仪的博客框架，推荐HEXO，或者是JeKyll，

    这两者都是用于生成伪静态页面的两个博客框架，这个博客是使用HEXO搭建的，接下来也是介绍HEXO的建站过程，如果想找JsKyll的框架，可以阅读大佬的博客https://faner.gitlab.io/blog/2017/07/07/jekyll%E5%AE%89%E8%A3%85%E4%B8%8E%E9%83%A8%E7%BD%B2/

  4. 安装HEXO的命令：npm install -g hexo-cli

    这一步可能会有warn，但是没事，只要没有error就不影响正常使用，如果发现npm无法请求到东西，可以尝试使用cnpm替换为国内源

  5. 接着输入：npm install hexo --save

  6. 安装完成后检查一下：hexo -v，如果输出正常，hexo就没问题了

  7. 这时在本地新建一个空白文件夹，加入命名为blog，作为你的博客的一个容器，博客相关的东西都会在这里，接着进入这个目录，输入：hexo init 完成hexo的初始化

  8. 接着输入hexo clean && hexo d && hexo s 把项目启动

  9. 点开 http://localhost:4000/ ，恭喜你！已经在本地搭建好博客了 (๑•̀ㅂ•́)و✧！距离成功只差下一步——部署博客到 GitHub Pages 了～

  10. 哦对了，还有一个东西，那就是Git，也要本地安装，有两种方式，一种是下载github客户端Github desktop，或者就是git的安装：可以从Git官网直接下载安装程序

配置 SSH key

要使用 git 工具首先要配置一下SSH key，为部署本地博客到 Github 做准备。

打开命令行输入 cd ~/.ssh 如果没报错或者提示什么的说明就是以前生成过的，直接使用 cat ~/.ssh/id_rsa.pub 命令就是可以查看本机上的 SSH key 了。
